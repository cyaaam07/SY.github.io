Tegelwip Atlas – Urban Greening Map

In this project, I built an interactive map of the Netherlands that visualizes areas of urban paving and potential greening. The map is responsive for both desktop and mobile use. Users can zoom into specific regions and view clear data visualizations that highlight where tiles can be replaced with green space. The goal was to make environmental impact more accessible and understandable through a clean and user friendly interface.

PHP HTML MySQL JavaScript CSS
